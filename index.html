O que é HTML

HTML (Hypertext Markup Language) é uma linguagem de marcação que define a estrutura e o conteúdo de uma página web.

Ele é o elemento básico da web e é usado para criar a estrutura de um site

O HTML é composto por tags, que são identificadas por sinais de menor e maior que (< >). As tags indicam o que cada parte do conteúdo significa

Tags

As tags HTML são elementos usados para estruturar e formatar o conteúdo de uma página web. Elas funcionam como "blocos de construção" que dizem ao navegador como exibir textos, imagens, links e outros conteúdos.

Atributo das Tags

Os atributos são informações extras que modificam o comportamento das tags HTML. Eles sempre aparecem dentro da tag de abertur

Links

No HTML, os links são criados com a tag 'a', que é usada para vincular uma página ou recurso externo. A principal função dessa tag é permitir a navegação de uma página para outra ou para uma seção específica dentro da mesma página.
